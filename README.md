# Nandgame Solution

Optimal solution for [Nandgame](https://www.nandgame.com/).

![screenshot](./.github/res/screenshot.png)

## Usage

- Click settings on nandgame.
- Click `Import` and paste text from `nandgame-data.json`.
- Click `Import JSON` and enjoy.
